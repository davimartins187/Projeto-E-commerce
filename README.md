# 🛒 Amazon Clone - E-commerce

Um projeto de e-commerce inspirado na Amazon, desenvolvido com **HTML, CSS e JavaScript**, com foco em simular as principais funcionalidades de uma loja virtual moderna.

## 📖 Sobre o Projeto

Este projeto foi criado para praticar desenvolvimento web front-end e conceitos de e-commerce, incluindo catálogo de produtos, carrinho de compras, gerenciamento de pedidos e rastreamento de entregas.

O sistema permite que os usuários naveguem pelos produtos, realizem compras, acompanhem seus pedidos e visualizem informações de entrega de forma intuitiva.

## ✨ Funcionalidades

### 🏠 Catálogo de Produtos
- Exibição de produtos
- Busca de itens
- Informações detalhadas dos produtos
- Layout responsivo

### 💳 Checkout
- Finalização de compras
- Resumo do pedido
- Cálculo de valores
- Confirmação de compra

### 📦 Pedidos
- Histórico de pedidos realizados
- Visualização dos detalhes da compra
- Status dos pedidos

### 🚚 Rastreamento
- Acompanhamento de entregas
- Visualização do progresso do pedido
- Informações de envio

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- JSON (armazenamento de dados)
